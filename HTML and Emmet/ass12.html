Comments

Comments are developer friendly,they are used to understand code better lateron.They are not displayed on the webpage,but they can be visible if anyone views the page source. Writing commets make code easier to understand and is useful while debugging.

Shortcut to comment a line is ctrl+/ .

Comments are of 3 types

  1. Single-line comment
  2. inline comments ,only the comments inside the comment tag gets commented.
  3. Multi-line comments
