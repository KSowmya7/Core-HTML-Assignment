Full Stack Web Development Tech Stack

HTML
HTML defines the basic layout and structure of a web page.It is simple and easy to understand and platform independent.
CSS
CSS is used to style the web page and to make it attractive.It describes the presentation of web pages,including colors,layouts and fonts.
Javascript
Javascript is used to make web page interactive,it is an interpreted language,case sensitive and platform independent.
Node.js
It is a cross-platform runtime environment that allows us to create server-side and networking applications.It is written in Javascript.
Express.js
Express is free and open source web application framework for Node.js,it is used for designing and buliding web applications quickly and easily.
React.js
ReactJS is a Javascript library used to create User Interfaces which enhance the speed of programs.Helps to create interactive and dynamic UI for mobile apps and websites.
MongoDB
MongoDB is an open source NoSQL database management program.It is a tool that can mange document-oriented informaton, store or retieve information.
