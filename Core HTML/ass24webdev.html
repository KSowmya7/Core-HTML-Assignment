Web Development

Web development,also known as website development,refers to the tasks associated with creating,builiding,and maintaining websites and web applications that run online on a browser.

There are 3 main types of web development

  1. Front-end development

    2. It refers to the portion of the site,app,or digital product that users will see and interact with them. The basic stools involved in front-end web development are HTML,CSS,Javascript.

  2. Back-end development

    3. Back-end development refers to the serverside development.It focuses on databases,scripting,website architecture.Backend developers are focused on how it works.Back-end use server-side programming languages,includig PHP,Python,Java,Node.js,Express.js,MongoDB etc.,

  3. Full stack web developmemt

    4. A full stack web developer can develop both client and server software.It is the process of designing,creatig,testing,and deploying a complete web application from start to finish.
