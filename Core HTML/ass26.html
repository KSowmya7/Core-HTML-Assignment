HTML Tags

  1. Image tag (img):It is used to insert an image in the document
  2. Paragraph tag (p): It is used to define paragraph of text,it adds content to the web page
  3. Header tags(h1 to h6): Used for headings,h1 has higher font,size reduces from h1 to h6
  4. Table tag(table): Defines a table
  5. Body tag(body): Defines the document's body
