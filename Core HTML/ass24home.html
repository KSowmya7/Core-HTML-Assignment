What is the main difference between Web Design and Web Development?

Web design is concered with what the user actually sees on their computer screen or mobile device, and less so about the mechanics beneath the surface thet makes it all work.

Web developers take the design created by the web designers and convert it into a fully functioning website.

Basically, web designer create a website's look and feel, and web developers create the code to make it work.
